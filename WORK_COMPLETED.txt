================================================================================
HOW TO RUN
================================================================================

Terminal 1 (Backend):
$ cd chat-backend
$ ./gradlew bootRun
# Backend runs on http://localhost:8080

Terminal 2 (Frontend):
$ cd chat-ui
$ npm install
$ npm run dev
# Frontend runs on http://localhost:5173

Then:
1. Open http://localhost:5173 in your browser
2. Type a message and press Enter
3. See user message appear (blue, right side)
4. See AI response appear (gray, left side)
